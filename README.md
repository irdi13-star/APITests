# API playwright tests

The API playwright tests is designed to verify the open source API gateway with well structure.

---

## Prerequisites

To run this project, make sure you have the following installed:

- Node.js

---

## Running the tests

1. Open the project:
   ```bash
   cd projectPath
   ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Run the tests:
    ```bash
    npm test
    ```

The API sources available at: 
    ```
    https://restful-booker.herokuapp.com/apidoc/index.html#api-Booking-DeleteBooking
    ```
